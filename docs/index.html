Accept Nano

Accept Nano is a payment gateway for NANO

Components

Server

You run accept-nano server for accepting NANO as a merchant.

Client

You use accept-nano client in your JS application to make requests to the server as the customer.

Demo

There is a working demo at https://rawgit.com/accept-nano/accept-nano-client/master/example/index.html
